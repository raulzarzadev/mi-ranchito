## Motivaciones

El Software puede contribuir a optimizar la producción agricola. Pero este debe ser económico y accesible

## Inicio Rapido

```

git clone git@github.com:raulzarzadev/mi-ranchito.git

cd mi-ranchito/

yarn dev

yarn build

yarn start

```

Variables de entorno ( Firebase Config )

```

PROJECT_ID

PRIVATE_KEY

CLIENT_EMAIL


```
